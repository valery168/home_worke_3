# Инструкция по работе с Git

# Команды 

для того , чтобы создать пустой репозиторий  нужно : 
> git init

что бы начать сохранение файла с именем file_name  
> git add file_name или  git add .

для создания комментария необходимо  
> git commit -m"...."

журнал всех коммитов  
> git log

для того что бы вернуться к старой версии необходимо:  
> git checkout

для возвра в актуальную версию  из старой версии:  
> git checkout master

для просмотра изменений необходимо:  
>git diff

для просмотра веток используй команду  
>git branch

для создания новой ветки используем команду плюс имя ветки   
>git branch branch_name

для переключения с ветки на ветку используем команду  git checkout и имя ветки куда хотим переключится
>git checkout branch_name

для слияния двух веток в одну необходимо перейти в ту ветку к которой необходимо что либо добавить и добавить ту ветку информацию из которой необходимо перелить используем команду "merge"  
>git checkout master 
>git merge branch_name

для удаления ветки используем команду "-d" дял этого набираем git branch -d и имя ветки .
>git branch -d barnch_name




# Оформление
для того чтобы создать заголовок необходимо поставить "#"  
># первый уровень
>## второй уровень
>### третий уровень
>#### четвёртый уровень

для выделения текста прописным шрифтом необходимо использовать "*" либо используя "_" в начале и в конце текста  
>*новый уровень* 

для выделения текста жирным шрифтом нужно использовать "**" либо двойного нижнего подчёркивания"__" в начале и в конце текста    
>**жирный текст**

для того чтобы написать жирным текстом курсивом используем "***"в начале и в конце текста    
***жирный текст курсивом***

для зачёркивания текста используй "~~" в начале и в конце текста  
>~~зачёркивание~~

для создания неномерованного спаиск используем "*" или "-" или "+" в начале текста  
>* элимент один
>- элимент два
>+ элимент три

для создания нумерованного списка используем "1" цифры перед тестом  
>1. элимент один
>2. элимент два
>3. элимент три

для создания вложенных пунктов нажми 4 раза на пробел перед пунктом 
>1. элимент    
    *подпункт

для создания цитаты используй ">" перед текстом
> цитата

для добавления картинки используем "!""[вводим имя картинки]""(название файла с расширением)"предварительно картинка должна быть размещена в папке с нашим visual studio. 
> ![bear](bear.jpg)

что бы создать новую ветку можно использовать укороченный вариант  
>git checkout -b branch_name

для быстрого сохранения можно использовать более короткую команду  
>git commit -a -m " comments"


что бы перейти к другой ветке используем 
>get checkout 




что бы удалить ветку которая была создана из не основной ветки используем  
>git branch -D branch_name

для того , что бы иметь возможность игнорировать какие либо файлы или надписи необходимо создать файл и поместить в него тот файл или информацию  которую  git будет игнорировать    
>.gitignore

для того что бы выгрузить из github информацию и произвести слияние её с предидущей если таковая была используем  
>git pull 

когда Вы авторизируйтесь наgithub вы получите возможность загружать данные на сервис github для загрузки информации на github необходимо 
воспользоваться командой push  
>git push

>git clone
